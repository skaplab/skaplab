📖 Teori

Ulikheter

En ulikhet sammenlikner to uttrykk med <, >, ≤ eller ≥. Løsningen er et intervall.

ax + b < c
📌 Symboler
< (mindre enn)   > (større enn)
≤ (mindre enn eller lik)   ≥ (større enn eller lik)
📖 Eksempel

Løs ulikhet

Løs: 2x + 3 < 11

2x < 8 → x < 4
📌 Viktig regel
Hvis du multipliserer eller dividerer med et negativt tall, snu ulikhets-tegnet!
📖 Teori

Heltallsløsninger

x < 4 betyr alle tall mindre enn 4. Heltallene er: ..., 1, 2, 3.

x ≤ n ⇒ x ∈ {..., n−1, n}
📌 Husk
x < 4: tallet 4 er ikke inkludert. x ≤ 4: 4 er inkludert.
