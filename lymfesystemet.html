Lymfe­systemet –
systema lymphaticum

Nettverket av kar og noder som returnerer vevsvæske, forsvarer kroppen mot infeksjoner og koordinerer immunresponsen

01 / 07
🔬 Oversikt

Lymfesystemets oppbygning

Lymfesystemet er et nettverk av kar, noder og organer som transporterer lymfevæske, forsvarer kroppen mot infeksjoner og fjerner overskuddsvæske fra vev. Det samarbeider tett med blodsystemet og immunsystemet.

💧
Lymfevæske (lymfe): Overskuddsvæske fra vev som er filtrert fra blodkapillærene. Inneholder hvite blodceller (lymfocytter), proteiner og avfallsstoffer. Transporteres i ett-veis retning mot hjertet.
🔀
Lymfekar: Åpent nettverk som starter i vevene som lymfehårkar. Samles til større lymfekar og til slutt thoraxkanalen (den største) og høyre lymfekanal. Tømmes i subklavievener ved hjertet.
⚙️
Drivkraft: Ingen hjertepumpe – lymfen drives fremover av skjellformet klaffer, muskelbevegelse og respirasjonsbevegelser. Stillsitting gir ødem (opphopning av vevsvæske).
🔢
Volum: 2–4 liter lymfevæske returneres til blodbanen per døgn. Lymfehårkar er 0,1–0,2 mm brede og absorberer bl.a. fett fra tynntarmen (kylus).
02 / 07
🔎 Filter

Lymfenoder – kroppens filtre

Lymfenodene er bønneformede strukturer (0,1–2,5 cm) spredt langs lymfekarene. De filtrerer lymfevæsken og aktiverer immunresponsen ved infeksjoner. Hevelse i lymfenodene er tegn på at immunsystemet er aktivt.

📍
Lokalisasjon: Særlig mange i lyske, armhuler, hals/nakke og tarm. Ca. 600–700 lymfenoder i hele kroppen. Grupperte noder i viktige drenasjesteder for lymfen fra ulike kroppsdeler.
🧱
Oppbygning: Ytre kapsely → Cortex (B-celler og follikler) → Paracortex (T-celler) → Medulla (makrofager og plasmaceller) → Hilus (lymfe-ut og blodårer). Afferente lymfekar inn, efferente ut.
⚔️
Immunforsvar: Makrofager fagocyterer bakterier og avfall. Dendritiske celler presenterer antigener for T-celler. B-celler aktiveres og produserer antistoffer. Naturlige dreperceller dreper virusinfiserte celler.
🔬
Lymfom: Kreft som starter i lymfenoder eller lymfevev. Hodgkin-lymfom (Reed-Sternberg-celler) og non-Hodgkin-lymfom. Svulne noder kan skyldes infeksjon, betennelse eller malignitet.
03 / 07
🫀 Organer

Primære og sekundære lymfoide organer

Lymfesystemet har en rekke spesialiserte organer. Primære organer produserer og modner lymfocytter; sekundære organer er der immunresponsen aktiveres mot spesifikke trusler.

🦴
Benmargen (primær): Produksjonssted for alle blodceller, inkludert B-lymfocytter. B-celler gjennomgår modning og seleksjon her. Umodne celler som reagerer på «selv» elimineres (sentraltoleransje).
⚗️
Tymus (primær): Bak brystbenet. T-lymfocytter modner og selekteres her. Positiv seleksjon: T-celler som gjenkjenner MHC overlever. Negativ seleksjon: selvreaktive T-celler elimineres. Atrofierer etter puberteten.
🍇
Milten (sekundær): Største lymfoide organ (~150 g). Filtrerer blod (ikke lymfe). Rød pulpa: fjerner gamle erytrocytter. Hvit pulpa: immunrespons mot blodbårne antigener. Lagrer trombocytter.
💎
Lymfoid vev (MALT): Mucosa-assosiert lymfoid vev. Tonsiller (svelg), Peyerske flekker (tynntarm), appendix. Første forsvarslinje mot antigener som trenger inn via slimhinner. IgA-sekresjon.
04 / 07
🦠 Celler

Lymfocytter – T-celler og B-celler

Lymfocyttene er de viktigste cellene i den adaptive immuniteten. T-celler styrer cellemediert immunitet; B-celler produserer antistoffer. Begge husker tidligere infeksjoner (immunologisk hukommelse).

🎯
T-hjelperceller (CD4+): Aktiverer B-celler og cytotoksiske T-celler. Produserer cytokiner (bl.a. interleukiner). Koordinerer immunresponsen. Målcelle for HIV-viruset, som derfor svekker hele immunforsvaret.
☠️
Cytotoksiske T-celler (CD8+): Dreper celler som er infisert med virus eller kreftceller. Gjenkjenner antigen på MHC klasse I. Friger perforin (danner hull) og granzym (induserer apoptose). Effektive mot intracellulære trusler.
💉
B-celler og plasmaceller: B-celler gjenkjenner antigener direkte eller via T-hjelperceller. Aktiverte B-celler differensierer til plasmaceller som produserer enorme mengder antistoffer (opptil 2 000 per sekund). Hukommelses-B-celler gir rask respons ved reinfekning.
🧬
NK-celler (naturlige dreper): Del av medfødt immunitet. Dreper celler med lavt eller manglende MHC-I uttrykk – tegn på virusinfeksjon eller kreft. Raskere enn T-celler, men mindre spesifikke.
05 / 07
💉 Immunitet

Antistoffer og immunologisk hukommelse

Antistoffer (immunglobuliner) er Y-formede proteiner produsert av plasmaceller. De binder spesifikke antigener og nøytraliserer trusler, merker dem for destruksjon eller aktiverer komplement-kaskaden.

🔑
Antistoffets struktur: To tunge + to lette kjeder (disulfidbindinger). Variabel region (Fab) = antigen-bindingssted – svært spesifikt. Konstant region (Fc) = effektorfunksjon – aktiverer komplement og fagocytter.
🏷️
Klasser (isotyper): IgG (vanligst, langvarig beskyttelse), IgM (tidlig respons), IgA (sekret og slimhinner), IgE (allergier og parasitter), IgD (B-celle-aktivering). Klasseskifte (isotype switching) reguleres av cytokiner.
🧠
Immunologisk hukommelse: Etter primær immunrespons dannes langlevede hukommelsesceller. Sekundær respons: raskere (1–3 vs 7–14 dager), sterkere, høyere antistofftitere. Grunnlag for vaksiner.
💊
Vaksinasjon: Gir aktiv immunitet uten sykdom. Levende svekket, inaktivert, subenhets-, toksoid- eller mRNA-vaksiner. mRNA-vaksiner (COVID-19) koder for antigen som kroppen selv produserer. Flokkimmunitet ved ~70–90% vaksinasjonsdekning.
06 / 07
🩺 Sykdom

Lymfeødem, lymfom og autoimmunitet

Svikt i lymfesystemet gir karakteristiske sykdomsbilder. Lymfeødem skyldes svekket drenasje; lymfom er kreft i lymfatisk vev; autoimmune sykdommer oppstår når immunsystemet angriper eget vev.

💧
Lymfeødem: Hevelse i vev pga. manglende lymfedrenasje. Primært (medfødt defekt) eller sekundært (kirurgi, stråling, infeksjon). Brystkreftpasienter mister noen ganger aksillære lymfenoder → armsødem. Elefantiasis: parasittisk blokkering (filarier).
🦠
Lymfom: Non-Hodgkin (B- eller T-celle-opprinnelse, mange undertyper) vs. Hodgkin (Reed-Sternberg-celler, bedre prognose). Symptomer: smertefri lymfeknutesvulst, nattsvette, vekttap, feber (B-symptomer). Behandles med kjemoterapi, stråling, immunterapi.
⚔️
Autoimmune sykdommer: Immunsystemet angriper eget vev. Lupus (anti-DNA), revmatoid artritt (ledd), type 1 diabetes (betaceller), MS (myelin). Behandles med immunsuppressiva. T-reguleringsceller (Treg) undertrykker normalt autoimmune responser.
🧪
Allergi: Overdrevet immunrespons mot ufarlige antigener (allergener). IgE binder mastceller → histaminfrigjøring. Anafylaksi: systemisk, livstruende histaminreaksjon. Behandles med antihistaminer, kortikosteroider, adrenalinstift (EpiPen).

📚 Oppsummering

Lymfesystemet – nøkkelpoenger

💧 TransportLymfehårkar → lymfekar → lymfenoder → thoraxkanal/høyre lymfekanal → blod. Klaffer hindrer tilbakestrøm. 2–4 L/døgn.
🔎 Noder og organer600–700 noder filtrerer lymfe. Tymus (T-celler), benmargen (B-celler) = primære. Milt, noder, MALT = sekundære lymfoide organer.
🦠 LymfocytterT-hjelperceller (CD4+) koordinerer. Cytotoksiske T (CD8+) dreper. B-celler → plasmaceller → antistoffer. NK-celler: medfødt dreper.
💉 AntistofferIgG/IgM/IgA/IgE/IgD. Fab = variabel (antigen-binding). Fc = effektorfunksjon. Immunologisk hukommelse = grunnlag for vaksiner.
🩺 SykdomLymfeødem (blokkert drenasje). Lymfom (Hodgkin/non-Hodgkin). Autoimmunitet (Treg svikt). Allergi: IgE + mastcelle → histamin.
🔗 SamarbeidLymfesystemet samarbeider med blod- og immunsystemet. Tarmens lymfekar (laktaler) absorberer fett. Milt renser blod og lagrer trombocytter.
